# Settlers

Settlers er 3D minimalist colony simulator. Þar sem þú byggir, stækkar og sérð um litla byggð. Settu niður hús, úthlutaðu verkefni og horfðu á byggðina stækka á meðan íbúarnir safna auðlindum, byggja hús og sjá um byggðina.

## Gameplay 

Leikmaður byrjar með litla byggð og fáa íbúa. Markmiðið er að byggja upp sjálfbæra nýlendu með því að:

- Safna auðlindum eins og stein, við og mat
- Stýra forgangsröð verkefnana
- Byggja byggingar eins og hús, námu, sveitir o.fl.
- Stækka byggðina smám saman

Leikurinn snýst um að hanna skilvirkt kerfi þar sem íbúar vinna saman og halda byggðinni gangandi.

## Core Systems

- **Task system:** Íbúar velja sjálfir verkefni eins og að safna, flytja eða byggja. Einnig væri hægt að manually setja vinnu á íbúa.
- **Pathfinding:** Íbuar get fært sig um bæinn sjálfvirkt án þess að fá hjálp frá Player
- **Resource System:** Byggingar sem framleiða og neyta auðlindir
- **Building system:** Byggingar eru settar á grid og þurfa auðlindir til að vera byggðar

## Features

- Top-down camera
- Grid-based building
- AI íbúar
- Sjá um auðlindir
- Stækkanleg byggð
- Minimalist aesthetic

## Markmið leiksins

Byggðu upp blómlega byggð með því að stjórna auðlindum og starfsmönnum á skilvirkan hátt. Gerðu tilraunir með staðsetningu og forgangsröðun verkefna til að skapa sjálfbæra byggð.
