# Settlers

Settlers er 3D minimalist colony simulator. Þar sem þú byggir, stækkar og sérð um litla byggð. Settu niður hús, úthlutaðu verkefni og horfðu á byggðina stækka á meðan íbúarnir safna auðlindum, byggja hús og sjá um byggðina.

## Features

- Grid-based building: byggðu hús auðveldu grid systemi
- Sjálfstæðir íbúar: AI sem sér um að safna auðlindum, byggja og færa dýrmæti.
- Sjá um auðlindir: Safnaðu við, stein og mat til að halda byggðini gangandi
- Stækkanleg byggð: Byrjaðu smátt og stækkaðu byggðina hægt og rólega.
- Minimalist aesthetic: Hægt að hugsa meira um leikinn sjálfan, heldur en útlitið.

## Markmið leiksins

Byggðu upp blómlega byggð með því að stjórna auðlindum og starfsmönnum á skilvirkan hátt. Gerðu tilraunir með staðsetningu og forgangsröðun verkefna til að skapa sjálfbæra byggð.
